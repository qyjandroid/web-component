# Toast

## 介绍

轻提示 Toast 组件

## 引入

```shell
import { Toast } from 'web-component';
```

## 指令式 API

Toast 只支持指令式调用。

## Toast.show

show 方法支持提示信息，提示展示时长（毫秒），提示展示类型。
